Welcome to SociAble:
Social Media Made Accessible

Social media has become an integral part of our lives, allowing us to connect with friends, family, and colleagues around the world. However, not all social media platforms are designed with accessibility in mind, and this can create barriers for users with disabilities. This website is to provide users with a comprehensive guide on accessible social media, bringing more awareness around the inclusion of disabled groups in social networking.

How To Use

Overall, to use the website, navigate using the links in the navigation bar to access the different sections of the website. Read through the information provided in each section to learn about how to make different social media platforms more accessible. Throughout the website, there are various accessibility features such as tabindex attributes and alternate text for images.

Twitter

Alternate Text:

Allows users to add alternative text to images for visually impaired users.

  1. After uploading an image to a Tweet, select Add description underneath the image.
  2. Enter a description of the image in the text box.
  3. Select Save. An ALT badge will appear in the corner of the image. (“ALT” is the abbreviation for alternative text, which is the technical name for image description.)
  4. Select Tweet. When someone selects the ALT badge, the image description appears on screen.

Keyboard Navigation:

Offers a way to navigate the website using only the keyboard for users who cannot use a mouse.

    Actions:

  • n = new Tweet
  • l = like
  • r = reply
  • t = Retweet

    Navigation:

  • ? = full keyboard menu
  • j = next Tweet
  • k = previous Tweet
  • space = page down

    Timelines:

  • g and h = Home timeline
  • g and o = Moments
  • g and n = Notifications tab
  • g and r = Mentions

Text Size and Color:

Provides options to adjust the text size and color contrast for users with visual impairments.

  1. Go to user settings.
  2. Go to Accessibility, display, and languages.
  3. Go to Display and sound.
  4. Change text size using the toggle. Turn on Dark mode.

Closed Captions:

Offers an option to include closed captions in videos for deaf or hard-of-hearing users.

  1. Upload a video to your Tweet.
  2. Select Upload caption file (.srt) below the video.
  3. In the pop-up, select the Upload captions button, then choose the caption file from your computer.
  4. Select Done.
  5. Under the video, the name of the language your captions are written in appears in place of the previous upload command.
  6. If you're satisfied with your Tweet text, video, and caption file, select Tweet. When your video plays, and the viewer has captions enabled, the captions will appear toward the bottom of the video.

In-App Contrast Adjustments:

Allows users to adjust the color contrast within the Twitter app itself, making it easier for users with visual impairments to access the content.

  1. Go to user settings.
  2. Go to Accessibility, display, and languages.
  3. Go to Accessibility.
  4. Under Vision, toggle Increase color contrast.

Instagram

Alternate Text:

Automatic alt text uses object recognition technology to provide a visual description of photos for people with visual impairments. You can replace this text to provide a better description of a photo. Keep in mind that this description will only be read if someone is using a screen reader to access Instagram.

    Editing Alt Text Before You Share:

  • Start by taking a photo or uploading an existing photo
  • Choose a filter and edit the image, then tap Next.
  • Tap Advanced Settings at the bottom of the screen.
  • Tap Write Alt Text.
  • Write your alt text in the box and tap Done.

    Editing Alt Text After You Share:

  • Go to the photo and tap
  • Tap Edit
  • Tap Edit Alt Text in the bottom right
  • Write the alt text in the box and tap Done

Voiceover Support:

Offers support for users with visual impairments by providing a voiceover feature that reads out the contents of the post and its associated comments.

  1. Go to user settings.
  2. Tap Accessibility, then tap Display & Text Size.
  3. Tap Larger Text.
  4. Tap the toggle to turn on Larger Accessibility Sizes.
  5. Drag the slider to adjust the text size.

Text Size and Color:

Provides options to adjust the text size and color contrast for users with visual impairments. You can adjust the text size of Instagram through your device or browser settings.

  1. Go to System preferences.
  2. Click Accessibility.
  3. Select VoiceOver from the left-side menu.
  4. Click to check the box next to Enable VoiceOver. You may need to confirm that you want to turn VoiceOver on.

Customizable Alt Text:

Automatic alt text uses object recognition technology to provide a visual description of photos for people with visual impairments. You can replace this text to provide a better description of a photo. Keep in mind that this description will only be read if someone is using a screen reader to access Instagram.

    Editing Alt Text Before You Share:

  • Start by taking a photo or uploading an existing photo
  • Choose a filter and edit the image, then tap Next.
  • Click Accessibility, then write the alt text in the box.
  • Click Share to post.

    Editing Alt Text After You Share:

  • Above your photo or video, click more options.
  • Tap Edit
  • Click Accessibility, then write the alt text in the box.
  • Click Done to save your changes.

Voiceover Gestures:

Offers a set of gestures that can be used to navigate the app using only voice commands, making it easier for users with physical disabilities to use the platform.

  1. Go to Settings > Accessibility > VoiceOver > VoiceOver Recognition, then turn on Image Descriptions.
  2. In an app such as Photos or Safari, select an image.
  3. Swipe up to hear more options, then double-tap when you hear “Explore image features.”
  4. Move your finger around on the image to find out the position of each object.
  5. You can add custom image descriptions with Markup..

TikTok

Captions and subtitles

Allows users to add captions or subtitles to their videos for deaf or hard-of-hearing users or non-native speakers.

  1. When uploading your video, tap the Captions button on the right side of the screen. TikTok will automatically transcribe any audio in the video.
  2. Once TikTok generates your caption, review it for accuracy. Tap the pencil icon on the bottom right corner of the screen.
  3. Tap save when you’re done!

Text-to-speech:

Offers a feature that reads out captions and comments for users with visual impairments.

  1. Tap the +, on the bottom of the screen, to create a video.
  2. Record or upload a video, then continue to the editing screen.
  3. Tap Text, on the bottom of the screen, then type out the text you would like to add to your video.
  4. Tap on the line of text you would like to be read out loud, then select Text-to-speech.
